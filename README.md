# EduSync: AI-Driven Academic Onboarding

## Project Overview
EduSync is an intelligent onboarding agent designed to enhance academic excellence through automated workflows. Developed during the AICW Fellowship (April 2026).

## Key Features
- Automated student/faculty onboarding.
- Grounded on academic excellence datasets.
- Built using Microsoft Copilot Studio.
